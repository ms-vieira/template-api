# template-api
Template de API Rest (Spring Boot) com teste unitário e tratamento de exceções, utilizando:
 - Gradlew 
 - Lombok
 - Spring Data JPA
 - Bean Validation
 - Controller Advice
 - JUnit
 - Hamcrest
 - Mockito 
 - h2database

Repositório do projeto: ->https://github.com/msvieiraweb/template-api

### Instruções para execução do projeto
- Clonar repositório.
- Importar o projeto em uma IDE
- Run As -> ``` WebStarter ```
